SCREEN5を使ったパレットアニメーションのサンプルです。
ラスタースクロールっぽいアニメーションの表現を狙っています。
005_PSEUDO_3D_3では奥行の表現（Y軸方向のスケール）がされていなかったので追加しました。

ファイル：
AUTOEXEC.BAS // プログラム本体
PSD3D4.SC5  // AUTOEXEC.BASの実行により生成されるパレットアニメーションのベースになる画像(SCREEN5)
