SCREEN5を使ったパレットアニメーションのサンプルです。
ラスタースクロールっぽいアニメーションを表現を狙っています。004_PSEUDO_3D_2ではSCREEN4のパレットアニメーションでしたが、
今回はSCREEN5にモードを変えました。結果として横8dots2色の制限がなくなり表現力が上がっています

ファイル：
AUTOEXEC.BAS // プログラム本体
PSD3D3.SC5  // AUTOEXEC.BASの実行により生成されるパレットアニメーションのベースになる画像(SCREEN5)
