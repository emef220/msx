SCREEN5を使ったパレットアニメーションのサンプルです。
スターウォーズのハイパースペース（ワープ航法）のようなアニメーションの表現を狙っています。
からの変更点は、RAND()関数の引数をTIMEから-TIMEに変更しています。

ファイル：
AUTOEXEC.BAS // プログラム本体
HYPER2.SC5  // AUTOEXEC.BASの実行により生成されるパレットアニメーションのベースとなる画像(SCREEN5)
