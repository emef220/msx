SCREEN4の表示サンプルです

ファイル：
AUTOEXEC.BAS // プログラム本体
PROGRAM.BIN  // SCREEN4画像データ(bin形式)
PROGRAM.ASM  // PROGRAM.BINを生成するためのdb文　(*1)

(*1) MSXPenのASMタブに入力しておくと、起動時にビルドしてPROGRAM.BINを生成してくれます
