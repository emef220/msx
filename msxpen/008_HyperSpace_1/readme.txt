SCREEN5を使ったパレットアニメーションのサンプルです。
スターウォーズのハイパースペース（ワープ航法）のようなアニメーションの表現を狙っています。

ファイル：
AUTOEXEC.BAS // プログラム本体
HYPER1.SC5  　// AUTOEXEC.BASの実行により生成されるパレットアニメーションのベースとなる画像(SCREEN5)
